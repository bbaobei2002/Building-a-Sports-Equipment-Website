Cell

Cell(float w [, float h [, string txt [, mixed border [, int ln [, string align [, boolean fill [, mixed link]]]]]]])

설명

셀(사각형 영역)을 출력합니다. 테두리 선이나 배경색 혹은 문자열을 동시에 설정할 수 있습니다. 셀의 좌측 상단을 현재 위치로 사용합니다. 문자열은 중간 등으로 정렬할 수 있습니다. 호출 후에, 현재 위치는 셀의 오른쪽 끝이나 다음줄로 이동합니다. 글자위에 링크를 설정할 수도 있습니다.
자동 페이지 바꿈이 활성화되어있는 상태에서 셀이 경계선을 넘었다면, 셀을 출력하기 전에 먼저 줄바꿈을 실행합니다.

파라미터

w
셀 너비. 0이면 셀은 오른쪽 끝까지 확장됩니다.
h
셀 높이. 기본 값은 0 입니다.
txt
출력할 문자열. 기본값 : 빈 문자열.
border
셀 주위에 그려질 경계선 지정. 기본값은 아래의 숫자 중 택일할 수 있습니다:
  • 0: 경계선 없음
  • 1: 프레임
혹은 아래의 문자나 문자의 조합으로 표현할 수 있습니다(순서는 관계없음):
  • L: 왼쪽
  • T: 위쪽
  • R: 오른쪽
  • B: 아래쪽
기본값: 0.
ln
호출후 좌표가 이동할 위치를 지정합니다. 가능한 값은 다음과 같습니다:
  • 0: 오른쪽
  • 1: 다음줄의 시작
  • 2: 아래쪽
1을 입력하는 것은 0을 입력후 바로 다음에 Ln()을 호출하는 것과 같습니다. 기본값: 0.
align
중간 등으로 텍스트를 정렬. 가능한 값은 다음과 같습니다:
  • L 혹은 빈 문자열: 왼쪽 정렬 (기본값)
  • C: 중간
  • R: 오른쪽 여백
fill
셀 배경색을 칠할 것인지(true) 투명으로 할 것인지(false)를 지정합니다. 기본값은 false 입니다.
link
URL 이나 AddLink() 실행후 반환된 고유값

예제

//글꼴 설정
$pdf->SetFont('Arial','B',16);
//8cm 오른쪽으로 이동
$pdf->Cell(80);
//20*10mm 크기의 셀에 중간정렬된 텍스트를 출력하고 줄 바꿈
$pdf->Cell(20,10,'Title',1,1,'C');
