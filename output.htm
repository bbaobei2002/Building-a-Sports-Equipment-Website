Output

string Output([string name, string dest])

설명

주어진 목적지 파라미터에 따라 문서를 저장하거나 웹브라우저로 전송합니다. 웹브라우저로 전송시에는 플러그인에 의해서 바로 출력이 되거나 "다른 이름으로 저장"을 할 수 있게 됩니다.
문서를 종료할 필요가 있다면 Close() 를 먼저 호출합니다.

파라미터

name
파일명. 입력하지 않으면 doc.pdf이란 이름으로 웹브라우저(목적지값 I)로 문서를 전송합니다.
dest
문서를 보낼 곳. 다음 값들 중 하나를 선택할 수 있습니다:
  • I: 파일을 브라우저로 보냅니다. 가능하다면 플러그인이 사용됩니다. 생성된 PDF를 저장하게 될 때엔 name에서 주어진 이름이 파일명으로 사용됩니다.
  • D: 브라우저로 보내고 name의 이름으로 강제로 다운로드하도록 합니다.
  • F: name에서 주어진 이름으로 로컬파일로 저장합니다.
  • S: 문서를 문자열로 반환합니다. name값은 사용하지 않습니다.

참고
