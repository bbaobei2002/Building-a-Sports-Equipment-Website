Rect

Rect(float x, float y, float w, float h [, string style])

설명

사각형을 출력합니다. 경계선만 그리거나(경계선만) 색을 채우거나(경계선 없이) 혹은 둘 다 가능합니다.

파라미터

x
좌측 상단 꼭지점의 가로 좌표
y
좌측 상단 꼭지점의 세로 좌표
w
너비
h
높이
style
출력 스타일. 가능한 값은 다음과 같습니다:
  • D 혹은 빈 문자열: 선그리기. 이것이 기본값입니다.
  • F: 색상 채우기
  • DF or FD: 선그리고 색상 채우기

참고

SetLineWidth(), SetDrawColor(), SetFillColor().
