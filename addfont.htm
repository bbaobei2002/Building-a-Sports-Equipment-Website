AddFont

AddFont(string family [, string style [, string file]])

설명

트루타입이나 Type 1 글꼴의 정보를 가져오거나 생성합니다. 글꼴 정의 파일을 최초로 생성할 때는 makefont.php 유틸리티가 필요합니다.
글꼴 정의 파일은(글꼴을 문서에 포함시킬때는 글꼴파일도) 반드시 글꼴이 있는 디렉토리에 존재해야 합니다. 만약 파일을 찾을 수 없다면, "Could not include font definition file(글꼴 정의 파일을 포함할 수 없음)" 오류가 발생합니다.

파라미터

family
글꼴 이름. 글꼴 이름은 자유롭게 지정할 수 있습니다. 표준 글꼴 이름을 사용한다면, 해당 글꼴 정보를 재정의합니다.
style
글꼴 스타일. 가능한 값은 아래와 같습니다(대소문자 구분안함):
  • 입력안함: 보통
  • B: 굵게
  • I: 기울게
  • BI or IB: 굵고 기울게
기본값은 보통 입니다.
file
글꼴 정의 파일
파일 이름을 지정하지 않으면 글꼴 이름과 스타일에서 자동으로 파일명(공백없는 소문자로)을 생성합니다.

예제

$pdf->AddFont('Comic','I');
위의 코드는 아래와 같습니다. 
$pdf->AddFont('Comic','I','comici.php');

참고

SetFont().
