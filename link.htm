Link

Link(float x, float y, float w, float h, mixed link)

설명

페이지의 사각영역에 링크를 출력합니다. 텍스트나 이미지 링크는 일반적으로 Cell(), Write(), Image() 등을 통해서 생성하지만 이 메소드는 이미지 내부의 일부 영역에만 간단히 링크를 걸 때 유용합니다.

파라미터

x
사각영역 좌측 상단 꼭지점의 가로좌표
y
사각영역 좌측 상단 꼭지점의 세로좌표
w
사각영역의 너비
h
사각영역의 높이
link
URL 이나 AddLink() 실행 후 반환된 고유값.

참고

AddLink(), Cell(), Write(), Image().
