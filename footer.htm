Footer

Footer()

설명

페이지 푸터를 만들때 사용합니다. AddPage() 와 Close() 메소드에 의해서 자동으로 호출되므로, 프로그램에서 직접 호출할 필요는 없습니다. FPDF 내에는 구현되지 않은채 존재하므로 별도의 처리를 원한다면 서브클래스를 구현해서 메소드를 오버라이드 해야합니다.

예제

class PDF extends FPDF
{
function Footer()
{
    //아래쪽에서 1.5 cm 떨어진 위치로 이동
    $this->SetY(-15);
    //기울어진 8크기의 Arial 선택
    $this->SetFont('Arial','I',8);
    //중간에 페이지 번호 출력
    $this->Cell(0,10,'Page '.$this->PageNo(),0,0,'C');
}
}
